## Running the API

Install JSON Server 

```
npm install -g json-server
```

Start JSON Server

```bash
$ json-server --watch db.json --port 3004
```

## Running the Frontend

Run the follow command

```
$ docker-compose up -d
```

Visit: http://localhost:3001/
